# 🛒 価格比較ツール かんたんガイド

## これは何？

楽天市場とモバイル一番（中古スマホ販売サイト）の**価格を自動で比較**して、
お得な商品を見つけてくれるツールです。

---

## 何ができるの？

```
例：iPhone 15を探している場合

📱 楽天市場: 98,000円
📱 モバイル一番: 89,800円
💰 差額: 8,200円！→ モバイル一番がお得！
```

**2,000円以上の差がある商品**を自動で見つけて、
Googleスプレッドシートに一覧を作ってくれます。

---

## 動かし方（3ステップ）

### ステップ1: 起動する

1. Docker Desktopを起動
2. ターミナル（黒い画面）を開く
3. 以下を入力してEnter：

```
cd Documents/n8n
docker compose up -d
```

4. 「Done」と出たらOK！

---

### ステップ2: 画面を開く

ブラウザ（ChromeやSafari）で以下を開く：

👉 **http://localhost:5678**

ログイン画面が出たら、メールアドレスとパスワードを入力

---

### ステップ3: 実行する

1. 左メニューの「Workflows」をクリック
2. 「楽天市場 vs モバイル一番 価格比較」を選択
3. 右上の **▶ Execute Workflow** ボタンを押す

これだけ！結果はGoogleスプレッドシートに出力されます。

---

## 止め方

ターミナルで以下を入力：

```
docker compose down
```

---

## よくある質問

### Q: パソコンを再起動したら動かなくなった

→ Docker Desktopを起動して、ステップ1からやり直し

### Q: エラーが出た

→ 以下のコマンドで再起動してみてください：
```
docker compose restart
```

### Q: 検索する商品を変えたい

→ n8nの画面で「🔧 検索パラメータ設定」をクリックして、`keyword`を変更
